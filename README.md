## Особенности

### Режим обучения (Одиночный элемент)
- Случайный выбор неизвестного элемента
- Визуализация TOF-спектра интенсивности
- График траекторий ионов в дрейфовой трубке
- Интерактивные подсказки при наведении
- Проверка догадок пользователя

### Режим анализа (Смеси газов)
- 5 предустановленных газовых смесей:
  - **Воздух** (N₂⁺, O₂⁺, Ar⁺, CO₂⁺)
  - **Углекислый газ** (CO₂⁺, N₂⁺, O₂⁺)
  - **Выхлопные газы** (CO⁺, CO₂⁺, NO⁺, NO₂⁺, SO₂⁺)
  - **Водород+Гелий** (H₂⁺, He⁺)
  - **Метан+Этан** (CH₄⁺, C₂H₆⁺, C₂H₄⁺)
- Визуализация соотношения интенсивностей
- Траектории с разной толщиной (пропорционально содержанию)
- Цветовое кодирование компонентов

### Визуальные возможности
- Адаптивный интерфейс для любых экранов (13"-27")
- Темная тема с контрастными элементами
- Два интерактивных графика:
  - **Спектр интенсивности** с референсными элементами
  - **Траектории ионов** с калибровочными линиями
- Автоматическое избегание пересечений подписей

### Интерактивность
- Регулировка ускоряющего напряжения (500-20000 В)
- Включение/выключение референсных элементов
- Показать/скрыть метки пиков
- Выделение ближайшего пика
- Подсказки при наведении на графики
